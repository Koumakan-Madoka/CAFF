指标说明 用于打磨提示词与工具链路（也可悬停查看 tooltip）
  • toolChatRate：监控 Agent 是否通过 send-public 等 public 工具把最终回复发到聊天室（turn 维度占比）。
  • privateToolRate：监控 Agent 是否按预期走 private 工具通道（turn 维度占比），用于排查私有输出/私聊指令是否生效。
  • send-public recall：提示词回归指标：当 expectations 标记 public 输出为 required 时，实际调用 public 工具的比例。
  • send-public fpr：提示词回归指标：当 expectations 标记 public 输出为 forbidden 时，仍调用 public 工具的比例（越低越好）。
  • send-private recall：提示词回归指标：当 expectations 标记 private 输出为 required 时，实际调用 private 工具的比例。
  • tool successRate：工具调用成功率；低通常是工具稳定性/参数问题（不一定是提示词问题）。
  • p50/p95：工具耗时分位数，用于定位慢工具、超时与重试。
